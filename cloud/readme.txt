Prerequisites:

Ensure that token authentication is enabled in your Splunk Cloud Platform instance. (See: https://docs.splunk.com/Documentation/SplunkCloud/latest/Security/EnableTokenAuth)
The user who downloads the script below must be a user with sc_admin privileges, not the user you created in the previous step.
You must have OpenSSL version 1.1.1n or higher.
You must have jq version 1.5 or higher.
